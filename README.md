# Checkers Game

A simple console-based Checkers game written in C.

## Files

```text
checkers/
├── chief.c
├── battlefield.h
├── battlefield.c
├── clanwar.h
├── clanwar.c
├── warrior.h
├── warrior.c
└── README.md
```

## Features

* Two-player game
* Piece movement
* Piece captures
* Multiple jumps
* King promotion
* Win detection
* Modular code structure

## Compile

```bash
gcc chief.c battlefield.c clanwar.c warrior.c -o checkers
```

## Run

```bash
./checkers
```

## Concepts Used

* Structures
* Arrays
* Functions
* Header files
* Modular programming
